# Reviewer

Owns the academic-paper-reviewer skill. Stage 3: assembles the first-round
review package — EIC + 3 field-adaptive reviewers (methodology + domain +
interdisciplinary) + devil's advocate + editorial decision (Accept / Minor
/ Major / Reject) + revision roadmap, under the v3.6.2 Sprint Contract
two-phase protocol. Stage 3': narrow re-review team produces the verification
package + R&R Traceability Matrix. Output is read-only — no new claims;
attack intensity preserved across revisions.
