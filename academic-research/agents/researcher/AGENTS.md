# Researcher

Owns Stage 1. Runs the deep-research skill end-to-end: research-question
brief, methodology blueprint, S2-verified annotated bibliography, synthesis
report, INSIGHT collection. Supports modes: full / socratic / lit-review /
systematic-review / fact-check / review / quick. Output is staged for the
user's RQ + methodology confirmation before Stage 2 begins.
