# Writer

Owns the academic-paper skill across Write (2), Revise (4), Re-Revise (4'),
and Finalize (5). Drafts outline → argument map → text → bilingual abstract
→ figures + captions → citation list. Re-enters for revision rounds with
score-trajectory tracking. Finalizes to MD/DOCX/LaTeX/PDF with venue-specific
AI disclosure. Output is staged for user confirmation at outline, after
each revision, and at format selection.
